# Разберем эту задачу по полочкам.
```

r = bin(n) #Переводим в двоичную степень
```

# Для корректного перевода числа, лишнее нужно убрать:
```
n = int(input())
r = bin(n) [2:] #Убираем лишние символы
print(r)
```
# Получаем:
### 50
### 110010

# Рассмотрим подбор цифр от 1 до 99
```
for n in range(1, 100): #Добавляем перебор чисел и переносим скрипт в цикл
    r = bin(n) [2:]
    if n % 2 == 0:
        r = r + '01'
    else:
        r = r + '10'
    print(n, r)
```
## Получаем результат с двоичной системе счисления:
### 1 110
### 2 1001
### 3 1110
### 4 10001
### 5 10110
### ...
## Но нам нужна десятичная:
```
for n in range(1, 100):
    r = bin(n) [2:]
    if n % 2 == 0:
        r = r + '01'
    else:
        r = r + '10'
    print(n, int (r, 2)) #Переводим ответ из двоичной в десятичную систему счисления
```
## Получаем десятичную систему счисления:
### 1 6
### 2 9
### 3 14

## Дальше по условию задания нам нужно найти минимальное число больше 130:
```
for n in range(1, 100):
    r = bin(n) [2:]
    if n % 2 == 0:
        r = r + '01'
    else:
        r = r + '10'
    if int(r, 2) > 130: #Условие для вывода - число больше 130
        print(int(r, 2)) #Вставляем вывод в условие
```

# Разберем следующие задание 
# 92) На вход алгоритма подаётся натуральное число N. Алгоритм строит по нему новое число R следующим образом: Строится двоичная запись числа N. К этой записи подписывается справа бит чётности: 0, если в двоичном коде числа N было чётное число единиц, и 1, если нечётное. К полученному результату подписывается ещё один бит чётности. Полученная таким образом запись (в ней на два разряда больше, чем в записи исходного числа N) является двоичной записью искомого числа R. Укажите минимальное число N, после обработки которого с помощью этого алгоритма получается число, большее, чем 184. В ответе это число запишите в десятичной системе.


## Начнем со скрипта для выполнения изначальных условий задания
```
n = int(input()) #Вводим число
r = bin(n)[2:] #Переводим число в двоичную степень счисления
print(r) #Выводим
if r.count('1') % 2 == 0: #Считаем кол-во единиц в строке и проверяем на четность 
    r = r + '0' #Если число четное - добавляем 0
else: 
    r = r + '1' #Если число НЕчетное - добавляет 1
print(r) #Добавляем первый бит четности
if r.count('1') % 2 == 0: 
    r = r + '0' 
else: 
    r = r + '1' 
print(r) #Добавляем второй бит четности
```

## Получаем:
### 50
### 110010
### 1100101
### 11001010

# Найдем ответ по условию: n у которого r > 184
```
for n in range (1, 100):
    r = bin(n)[2:]
    if r.count('1') % 2 == 0:
        r = r + '0'
    else:
        r = r + '1'
    if r.count('1') % 2 == 0:
        r = r + '0'
    else:
        r = r + '1'
    if int(r, 2) > 184: #Выводи n, если r больше 184
        print(n)
```

# Следующее задание
# Напишите программу, которая ищет среди целых чисел, принадлежащих числовому отрезку [126849; 126871], числа, имеющие ровно 4 различных делителя. Выведите эти четыре делителя для каждого найденного числа в порядке возрастания.

## Напишем программу для нахождения кол-ва делителей
```
x = 24 #Пример числа
c = 0 #Счетчик кол-ва делителей
for i in range(1, x+1): #Перебор чисел для деления
    if x % i == 0: #Проверка
        c += 1 # +1 за каждый найденый делитель
print(c) # Вывести число делителей
```
## Вывод:
### 8

```
for x in range(126849, 126872): #Перебор чисел по условию
    c = 0
    for i in range(1, x + 1):
        if x % i == 0:
            c = c + 1
    print(x, c) #Вывод числа и кол-ов его дителелей
```
