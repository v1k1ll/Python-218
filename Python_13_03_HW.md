# Разберем эту задачу по полочкам
```
n = int(input()) #Вводим число
r = bin(n) #Переводим в двоичную степень
print(r)
```
# Вывод - у нас выводится строка с лишними символами: 
### 50
### 0b110010
# Для корректного перевода числа, лишнее нужно убрать:
```
n = int(input())
r = bin(n) [2:] #Убираем лишние символы
print(r)
```
# Получаем:
### 50
### 110010
----------------------------------------------------------------------

# Рассмотрим подбор цифр от 1 до 99
```
for n in range(1, 100): #Добавляем перебор чисел и переносим скрипт в цикл
    r = bin(n) [2:]
    if n % 2 == 0:
        r = r + '01'
    else:
        r = r + '10'
    print(n, r)
```
## Получаем результат с двоичной системе счисления:
### 1 110
### 2 1001
### 3 1110
### 4 10001
### 5 10110
### ...
## Но нам нужна десятичная:
```
for n in range(1, 100):
    r = bin(n) [2:]
    if n % 2 == 0:
        r = r + '01'
    else:
        r = r + '10'
    print(n, int (r, 2)) #Переводим ответ из двоичной в десятичную систему счисления
```
## Получаем десятичную систему счисления:
### 1 6
### 2 9
### 3 14
### 4 17
### 5 22
### ...
--------------------------------------------

## Дальше по условию задания нам нужно найти минимальное число больше 130:
```
for n in range(1, 100):
    r = bin(n) [2:]
    if n % 2 == 0:
        r = r + '01'
    else:
        r = r + '10'
    if int(r, 2) > 130: #Условие для вывода - число больше 130
        print(int(r, 2)) #Вставляем вывод в условие
```
### ОТВЕТ: *134*

## Напишем программу для нахождения кол-ва делителей
```
x = 24 #Пример числа
c = 0 #Счетчик кол-ва делителей
for i in range(1, x+1): #Перебор чисел для деления
    if x % i == 0: #Проверка
        c += 1 # +1 за каждый найденый делитель
print(c) # Вывести число делителей
```
## Вывод:
### 8

## Нам нужно вывести сами делители
```
for x in range(126849, 126872):
    c = set() #Множество
    for i in range(1, x + 1):
        if x % i == 0:
            c.add(i) #Добавить в множество
    if len(c) == 4: #Если кол-во записей в множесте = 4
        print(c) #Вывод делителей
```
